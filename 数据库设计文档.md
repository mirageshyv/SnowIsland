## 2. 数据表结构设计

### 2.1 用户表（user）
| 字段名 | 数据类型 | 长度 | 约束 | 说明 |
|--------|----------|------|------|------|
| id | INT | 11 | PRIMARY KEY, AUTO_INCREMENT | 用户唯一标识 |
| username | VARCHAR | 50 | NOT NULL, UNIQUE | 登录账号 |
| password | VARCHAR | 100 | NOT NULL | 密码（明文存储） |
| role | ENUM | - | NOT NULL | 用户权限：'dm'（游戏主持人）、'player'（玩家） |
| player_id | INT | 11 | FOREIGN KEY, NULL | 关联玩家表ID（仅玩家角色有值） |
| created_at | DATETIME | - | NOT NULL, DEFAULT CURRENT_TIMESTAMP | 创建时间 |
| updated_at | DATETIME | - | NOT NULL, DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP | 更新时间 |
| status | TINYINT | 1 | NOT NULL, DEFAULT 1 | 账号状态：1-正常，0-禁用 |

### 2.2 玩家表（player）
| 字段名 | 数据类型 | 长度 | 约束 | 说明 |
|--------|----------|------|------|------|
| id | INT | 11 | PRIMARY KEY, AUTO_INCREMENT | 玩家唯一标识 |
| name | VARCHAR | 50 | NOT NULL | 玩家姓名 |
| is_weak | TINYINT | 1 | NOT NULL, DEFAULT 0 | 虚弱状态：1-是，0-否 |
| is_overworked | TINYINT | 1 | NOT NULL, DEFAULT 0 | 过劳状态：1-是，0-否 |
| is_injured | TINYINT | 1 | NOT NULL, DEFAULT 0 | 受伤状态：1-是，0-否 |
| job_id | INT | 11 | FOREIGN KEY, NOT NULL | 职业ID（关联职业表） |
| skill_id | INT | 11 | FOREIGN KEY, NULL | 个人技能ID（关联个人技能表） |
| faction | ENUM | - | NOT NULL | 阵营：'统治者'，'反叛者'，'冒险者'，'杀戮者'，'平民' |
| created_at | DATETIME | - | NOT NULL, DEFAULT CURRENT_TIMESTAMP | 创建时间 |
| updated_at | DATETIME | - | NOT NULL, DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP | 更新时间 |

### 2.3 职业表（job）
| 字段名 | 数据类型 | 长度 | 约束 | 说明 |
|--------|----------|------|------|------|
| id | INT | 11 | PRIMARY KEY, AUTO_INCREMENT | 职业唯一标识 |
| name | VARCHAR | 50 | NOT NULL, UNIQUE | 职业名称 |
| skills | TEXT | - | NOT NULL | 职业具体技能描述 |
| created_at | DATETIME | - | NOT NULL, DEFAULT CURRENT_TIMESTAMP | 创建时间 |
| updated_at | DATETIME | - | NOT NULL, DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP | 更新时间 |

### 2.4 个人技能表（skill）
| 字段名 | 数据类型 | 长度 | 约束 | 说明 |
|--------|----------|------|------|------|
| id | INT | 11 | PRIMARY KEY, AUTO_INCREMENT | 技能唯一标识 |
| name | VARCHAR | 50 | NOT NULL, UNIQUE | 技能名称 |
| function | TEXT | - | NOT NULL | 技能具体功能描述 |
| created_at | DATETIME | - | NOT NULL, DEFAULT CURRENT_TIMESTAMP | 创建时间 |
| updated_at | DATETIME | - | NOT NULL, DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP | 更新时间 |
