# 覆雪之下 - 游戏玩家物资管理系统测试报告与使用说明

## 1. 系统概述
本系统是为"覆雪之下"线上游戏交互网站设计的玩家物资管理系统，用于管理玩家拥有的各类物资，包括道具、武器、子弹和基础物资。

## 2. 系统功能
- **道具管理**：展示玩家拥有的所有道具类物资
- **武器与子弹管理**：展示玩家拥有的所有武器及对应的子弹
- **基础物资管理**：展示玩家拥有的所有基础物资
- **数据可视化**：通过表格形式清晰展示物资信息
- **空状态处理**：当玩家没有某类物资时显示提示信息

## 3. 数据库设计
- **表结构**：包含道具表、武器表、子弹表、物资表和玩家物资关联表
- **关系**：玩家与各类物资通过player_items表建立多对多关联
- **索引**：为关键字段创建索引，提高查询性能

## 4. 前端实现
- **技术栈**：Vue 3 + Element Plus + Vue Router
- **页面结构**：
  - 物资管理主页面：包含左侧导航栏和右侧内容区
  - 道具类别页面：展示玩家拥有的道具
  - 武器与子弹类别页面：展示玩家拥有的武器和子弹
  - 物资类别页面：展示玩家拥有的基础物资
- **响应式设计**：适配不同屏幕尺寸

## 6. 使用说明

### 6.1 系统访问
1. 启动前端服务：`npm run dev`
2. 启动后端服务：`mvn spring-boot:run`
3. 访问登录页面：http://localhost:5173/
